# AI Assisted Development Note

During development, I used ChatGPT as an AI-assisted development tool to help design and implement the User Management System. The tool assisted in creating Flask routes, integrating SQLite database operations, implementing password hashing, validating user inputs, and improving the user interface using Bootstrap.

AI assistance was also used for troubleshooting environment setup issues, debugging login functionality, handling form validation, and improving code organization. One challenge encountered during development was resolving authentication and routing issues, which were identified and fixed through iterative testing and debugging with AI guidance.
